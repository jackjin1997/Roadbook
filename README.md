## Features

| Feature | Description |
|---------|-------------|
| **Living Skill Graph** | Global D3 radial force graph вҖ” your entire skill map on the home page |
| **Skill Decay** | Skills fade over time based on `lastActiveAt` вҖ” see what you're forgetting |
| **JD Match** | Paste a job description, get a match score with mastered/learning/missing breakdown |
| **Skill Timeline** | Activity feed showing every skill status change across all workspaces |
| **Share Card** | Radar chart skill card with PNG export вҖ” your tech fingerprint |
| **Multi-source Journey** | Upload JDs, articles, PDFs вҖ” Ariadne merges them into one unified skill roadmap |
| **RAG Chat** | Vector-store retrieval-augmented Q&A, responds in your language (zh/en/ja/es/fr) |
| **Research Todo** | AI auto-researches topics online, generates research sources |
| **Graph / Prose** | Toggle between radial force graph and Markdown rendering |
| **Obsidian Export** | One-click `.zip` vault with `[[wikilinks]]` between skill nodes |
| **LangSmith Tracing** | Full observability on every generation step |

- **Radial force layout** вҖ” `d3.forceRadial` places High/Medium/Low skills on concentric rings
- **Skill Decay** вҖ” node opacity fades from 1.0 to 0.3 over 90 days of inactivity
- **Expandable sub-skills** вҖ” click a node to reveal satellite sub-skill nodes
- **Status tracking** вҖ” double-click to cycle Not Started вҶ’ Learning вҶ’ Mastered
- **MiroFish interactions** вҖ” highlight selection, connected-node focus
- **Dopamine Max theme** вҖ” dual dark/light mode with 10-color category palette and glow effects

## Quick Start

```bash
pnpm install
cp .env.example .env   # fill in API keys
pnpm dev               # вҶ’ http://localhost:1420
```

### Environment Variables

```env
GOOGLE_API_KEY=          # Gemini (default model)
ANTHROPIC_API_KEY=       # Claude (optional)
OPENAI_API_KEY=          # GPT (optional)
TAVILY_API_KEY=          # required вҖ” web research
LANGSMITH_API_KEY=       # optional вҖ” tracing
```

## Design

**Dopamine Max** вҖ” dual-theme design system (see [DESIGN.md](./DESIGN.md) for full spec):

- **Dark mode** (primary): deep purple-black `#0A0A12` canvas, neon glow nodes, glassmorphism cards, ambient gradient blobs
- **Light mode**: warm cream `#FFFBF5` canvas, soft pastel nodes with white halos, warm brown borders
- **Typography**: Satoshi (hero), Plus Jakarta Sans (body), JetBrains Mono (brand/data)
- **10-color dopamine palette** per theme вҖ” each skill category gets a dedicated vibrant color
- **Motion**: learning pulse, mastered breathe, unlock burst animations
- Graph layout inspired by [MiroFish](https://github.com/666ghj/MiroFish)
